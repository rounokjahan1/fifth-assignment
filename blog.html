Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a tree-like representation of a web page's structure, where each element is a node. It allows developers to manipulate content and elements using JavaScript. This makes it easy to change text, add elements, or respond to user actions dynamically.

How do you select an element from the DOM?

I can select an element from the DOM using these methods:

1. By ID: document.getElementById('id') selects an element with a specific ID.

2. By class: document.getElementsByClassName('class') selects elements with a certain class.

3. By CSS selector: document.querySelector('selector') selects the first element that matches the given CSS selector.

4. To select multiple elements with a CSS selector: document.querySelectorAll('selector') selects all elements in the DOM that match the specified CSS selector and returns them as a NodeList.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that involves adding a single event listener to a parent element, allowing it to handle events for all its child elements. When a child element triggers an event, it bubbles up to the parent, which can process it. This method improves performance, reduces code complexity, and automatically works with dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM:

1. Change Attributes: Use element.setAttribute('attributeName', 'value') to change an attribute.

2. Get Attributes: Use element.getAttribute('attributeName') to retrieve an attribute's value.

3. Change Styles: Use element.style.propertyName = 'value' to change a specific style.

4. Add/Remove Classes: Use element.classList.add('className') to add a class or element.classList.remove('className') to remove one.
